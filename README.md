The Odin Project - Project Recipes 

## Project contains:

- **HTML**: Structure of pages and basic markup.  
- **CSS**: Styling of pages and responsive design.  
- **Images**: Visual representations of dishes and recipes.  
- **Links**: Navigation between pages (Home, Recipes, Add Recipe).

---

## Features

- View recipes with details about ingredients and preparation steps.  
- Add new recipes.  
- Responsive design that adapts to mobile devices.  
- Simple navigation and clear layout.
